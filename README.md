# Healthcare Patient Analytics Dashboard

## Project Overview
Analyzed 55,500+ hospital patient records to identify 
billing patterns, medical condition trends, and 
insurance provider performance using PostgreSQL & Power BI.

## Tools Used
- PostgreSQL (Database Design, Complex Queries, 
  Window Functions, CTEs)
- Power BI (DAX Measures, Data Modeling, 
  Patient Risk Simulator)

## Key Insights
- Total Hospital Revenue: $1.42 Billion
- Avg Billing per Patient: $25,539
- Avg Hospital Stay: 15.51 days
- Obesity highest avg billing: $25,806
- Asthma longest stay: 15.70 days
- 2020 peak admissions year

## Dashboard Pages
- Page 1: Hospital Patient Overview
- Page 2: Medical Condition Analysis
- Page 3: Insurance & Billing Analysis
- Page 4: Patient Risk & Billing Simulator
